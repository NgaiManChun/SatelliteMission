# SatelliteMission
"Satellite Mission" developed by Ngai Man Chun

Unityで制作した3Dアクションシューティングゲームです。  
プレイヤーは自機の周囲を飛び回る「サテライトライフル」を駆使しながら、
敵軍を殲滅し、自軍レーダーを防衛します。

実行ファイルは[こちら(https://drive.google.com/file/d/1hvP8K220n4-ZvYPq6GzIuPjMlpQGE0No/view?usp=sharing)]

# 操作方法

## キーボード・マウス

| 操作 | 内容 |
|---|---|
| WASD | 移動 |
| Space | 上昇 |
| マウス | 視点操作 |
| 左クリック | メイン武器（サテライトライフル） |
| 右クリック | サブ武器（ミサイル） |
| 左Shift | 照準アシスト ON/OFF |
| Enter | 一時停止 |

---

## コントローラー

| 操作 | 内容 |
|---|---|
| 左スティック | 移動 |
| 右スティック | 視点操作 |
| ZL | 上昇 |
| R | メイン武器 |
| ZR | サブ武器 |
| X | 照準アシスト ON/OFF |
| + | 一時停止 |

※ Nintendo Switch Pro Controller基準

---

# 勝利条件 / 敗北条件

## 勝利条件
- 敵を全滅する

## 敗北条件
- 自機が破壊される
- 自軍レーダーが破壊される

---

# 武器

## サテライトライフル
自機の周囲を不規則に旋回する武装です。  
攻撃時はターゲット方向へ展開し、自動連射を行います。

## ミサイル
追尾性能を持つサブ武器です。  
一定時間ごとにターゲット方向を補正しながら飛行します。
